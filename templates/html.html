HTML Introduction

HTML (HyperText Markup Language) is the standard markup language used to create and design documents on the World Wide Web. It provides the structure and content of a web page by using a system of tags and attributes.

  • Tags: HTML uses tags to define elements within a document. Tags are enclosed in angle brackets <> and typically come in pairs: an opening tag and a closing tag. For example, <p> is the opening tag for a paragraph, and </p> is the closing tag.
  • Attributes: Tags can have attributes that provide additional information about the element. Attributes are placed within the opening tag and are usually in the form of name-value pairs. For example, the <a> tag (anchor tag) has an href attribute that specifies the URL of the link.
  • Elements: Elements are made up of tags, content, and attributes. They define the structure and content of a web page. For example, the <h1> element defines a top-level heading, and the <img> element embeds an image in the page.
  • Document Structure: A typical HTML document has a structure that includes a <!DOCTYPE html> declaration at the beginning to indicate the document type, followed by an <html> element that contains <head> and <body> sections. The <head> section includes meta-information about the document, such as the page title and links to stylesheets. The <body> section contains the main content of the document.
  • Semantic HTML: Semantic HTML refers to using HTML elements that convey meaning about the content they contain. For example, using <header> for a page header, <nav> for navigation links, <section> for a section of content, and <footer> for a page footer.
  • HTML5: HTML5 is the latest version of HTML and introduces new elements, attributes, and features that improve the functionality and accessibility of web pages. It includes elements like <video>, <audio>, <canvas>, and more.

HTML Link

In HTML, a link is created using the <a> (anchor) tag. The <a> tag is used to define a hyperlink, which is a clickable text or image that redirects the user to another location.

  <a href="https://www.example.com">Click here to visit Example.com</a>

CSS Link

In HTML, you can link an external CSS stylesheet to style your web page using the <link> tag. Here's an example:

  <!DOCTYPE html>
  <html>
  <head>
    <title>My CSS Example</title>
    <link rel="stylesheet" type="text/css" href="styles.css">
  </head>
  <body>
    <h1>Hello, CSS!</h1>
    <p>This is a paragraph styled with CSS.</p>
  </body>
  </html>

HTML Tags Example

Practical example of using various HTML tags in a simple webpage:

  <!DOCTYPE html>
  <html>
  <head>
    <title>HTML Tags Example</title>
    <style>
      body {
        font-family: Arial, sans-serif;
        margin: 20px;
        padding: 0;
      }
      h1 {
        color: #333;
      }
      p {
        color: #666;
      }
      ul {
        color: #999;
      }
      .important {
        font-weight: bold;
        color: #f00;
      }
    </style>
  </head>
  <body>
    <h1>Welcome to My Website</h1>
    <p>This is a paragraph of text.</p>
    <p class="important">This is an important paragraph.</p>
    
    <h2>Lists</h2>
    <ul>
      <li>Item 1</li>
      <li>Item 2</li>
      <li>Item 3</li>
    </ul>
    
    <h2>Links</h2>

    <img src="https://via.placeholder.com/150" alt="Placeholder Image">
    
    <h2>Tables</h2>

        <th>Name</th>

      </tr>

      </tr>

  </html>

Navbar Example

Here's a basic example of a navbar in HTML:

  <!DOCTYPE html>
  <html>
  <head>
    <title>Navbar Example</title>
    <style>
      body {
        font-family: Arial, sans-serif;
        margin: 0;
        padding: 0;
      }
      .navbar {
        background-color: #333;
        overflow: hidden;
      }
      .navbar a {
        float: left;
        display: block;
        color: white;
        text-align: center;
        padding: 14px 20px;
        text-decoration: none;
      }
      .navbar a:hover {
        background-color: #ddd;
        color: black;
      }
    </style>
  </head>
  <body>

  <div class="navbar">
    <a href="#home">Home</a>
    <a href="#about">About</a>
    <a href="#services">Services</a>
    <a href="#contact">Contact</a>
  </div>

  <div style="padding:20px">
    <h2>Navbar Example</h2>
    <p>A basic example of a navbar in HTML.</p>
  </div>

  </body>
  </html>
